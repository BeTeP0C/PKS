# PKS

Консольный калькулятор на C# с поддержкой операций:
+ - * / % 1/x x² √x M+ M- MR

## Запуск

1. Установите [.NET SDK](https://dotnet.microsoft.com/download).
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/BeTeP0C/PKS.git
   cd PKS
   dotnet run

## 1) Функционал

Установка текущего значения: вводом числа (поддерживает как 12.5, так и 12,5).

Бинарные операции:
+ n, - n, * n (или x n), / n (или ÷ n), % n (остаток от деления).

Унарные операции:
inv/1/x (обратить), sq/x^2 (квадрат), sqrt (квадратный корень).

Память:
M+ — прибавить текущее к памяти,
M- — вычесть текущее из памяти,
MR — записать значение памяти в текущее.

Сбросы:
C — текущего значения, AC — текущего значения и памяти.

Сервис: help — помощь, exit — выход.

После каждой команды — отображение состояния: Текущее / Память.

## 2) Ограничения

Тип хранения — double (плавающая точка): возможны небольшие погрешности представления (как у всех калькуляторов на double).

Операция % реализована как остаток от деления для double (матем. mod). Это не «процент от», а именно остаток.

Переполнения при очень больших значениях ловятся и сообщаются как ошибка.

Локаль числа: принимаются и точка, и запятая, но вывод идёт в локали вашей ОС.

## 3) Возможные ошибки

Деление на ноль: / 0, inv при текущем 0, % 0 — выдаст сообщение «деление на ноль».

Корень из отрицательного: sqrt при current < 0.

Переполнение: при крайне больших промежуточных результатах (OverflowException).

Отсутствует аргумент: для бинарных операций без числа (например, просто +) — будет подсказка о необходимости аргумента.

Неверный ввод: если число распарсить не удалось — сообщение об ошибке.
